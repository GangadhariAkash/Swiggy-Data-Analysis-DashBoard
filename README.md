# 🍽 Food Delivery Sales & Performance Dashboard (Excel)

## 📌 Project Overview
This project is an interactive **Excel dashboard** created to analyze food delivery data and generate business insights using Pivot Tables, Pivot Charts, and Slicers.

## 📊 Key Metrics
- Total Revenue
- Average Order Value
- Average Customer Rating
- Total Orders

## 📈 Dashboard Insights
- Orders by Food Category
- Monthly Order Trend
- Top Restaurants by Revenue
- Top 10 Popular Dishes

## 🛠 Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Data Cleaning & Formatting
- Dashboard Design

## 🎯 Business Value
This dashboard helps stakeholders:
- Identify high-revenue restaurants
- Understand customer food preferences
- Track monthly demand trends
- Make data-driven decisions
